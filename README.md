# coding-reboot

A personal coding reboot repository for refreshing programming fundamentals, rebuilding consistency, and strengthening practical development skills through daily practice.

## Purpose
This repository is meant to help me get back into coding after a long break by focusing on:
- JavaScript fundamentals
- Git and GitHub workflow
- terminal confidence
- debugging practice
- small hands-on exercises
- mini projects and coding challenges

## What this repo will contain
- daily practice files
- short coding exercises
- notes and reminders
- small projects
- experiments with code
- progress updates and learning milestones

## Goal
The goal is not just to review code, but to become comfortable building again with confidence, structure, and consistency.

## Learning path
- Week 1: JavaScript fundamentals
- Week 2: Mini projects
- Week 3+: Real project building
