What is a function? 
Answer: A function in JavaScript is a named block of code that does a job. A function can also be said to be a reusable piece of code that performs a specific task.
What is the difference between creating and calling a function?
Answer: Simply put, creating a function means writing it.
Calling a function means running it.
What is a parameter?
Answer: A parameter can be said to be a value a function expects to receive.
What does return do?
Answer: The word return sends a value back from the function.
When would you use a function instead of repeating code?
Answer: A function is used when there is a need for the same code more than once, or when one wants to keep a code cleaner and easier to understand.

For example, instead of writing the same greeting code over and over, one will write it once in a function and reuse it whenever needed.

Using function saves time, reduces mistakes, and makes the code easier to update.
